INTRODUCTION TO OPERATING SYSTEMS

OPERATING SYSTEMS

IT acts as an interface between the user and the hardware. it enables user to communicate and manage computer hardware and software.

it controls:

  1. Files (update,create,access rights).
  2. Devices (i/o devices,hard-disks,memory).
  3. processing time in a multi user environment.
  4. users on how to use the system:
  • window family
  • linux/UNIX
  • DOS
  • Novell Netware
  • sun solaris
  • Macintosh

OS Input/output Software Layers

Basically, input/output software organized in the following four layers:

  1. Interrupt handlers
  2. Device drivers
  3. Device-independent input/output software
  4. User-space input/output software

In every input/output software, each of the above given four layer has a well-defined function to perform and a well-defined interface to the adjacent layers. The figure given below shows all the layers along with hardware of the input/output software system.
